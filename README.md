# Premier programme

## Private Chat

**Description**

*Création d'un chat privé (client/serveur) afin de s'échanger toutes sortes d'informations.*

**Compilation**

Serveur : `gcc -o server.c server -O0`

Client : `gcc -o client.c client -O0`

**Utilisation Serveur**

`./server`

Port : 1025

**Utilisation Client**

`./client`

Port : 1025

**Quitter les programmes**

`Ctrl + C`
